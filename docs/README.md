  * 입력받은 자동차 이름이 유효한지 검증하는 메서드 ㅇ
  * 사용자로부터 시도 횟수를 입력받아 반환하는 메서드ㅇ
  * 주어진 자동차 리스트에 대해 무작위 값에 따라 전진 여부를 결정하고 업데이트 하는 메서드
  * 현재 자동차들의 위치를 출력하는 메서드
  * 우승자를 결정하여 반환하는 메서드
  * 우승자를 알리는 메시지를 출력하는 메서드
